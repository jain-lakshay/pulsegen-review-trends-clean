# Pulsegen Review Trends

Pulsegen Review Trends is a Python-based project designed to collect, process, and analyze user reviews from mobile applications on the Google Play Store. The system extracts key insights, identifies trending topics, and visualizes sentiment trends over time.

---

## Features

- **Automated Review Collection:** Fetches reviews from Google Play Store with configurable date ranges and limits.  
- **Text Preprocessing:** Cleans and prepares raw review text for analysis.  
- **Embeddings Generation:** Uses OpenAI or Sentence Transformers to convert reviews into vector representations.  
- **Trend Analysis:** Extracts trending topics from reviews and generates daily/30-day trends.  
- **CSV Output:** Generates CSV files for raw reviews and analyzed trends for further use.
